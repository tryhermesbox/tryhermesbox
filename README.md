# Hermes Box

Managed cloud hosting for Hermes Agent.

Hermes Box gives every user a personal AI assistant running on dedicated cloud infrastructure. Pay with SOL, deploy a server, connect your messenger, and let Hermes Agent work from a real machine instead of a browser tab.

## What Hermes Box Does

- Deploys hosted Hermes Agent instances from a wallet-first flow
- Accepts SOL payment for dedicated server deployment
- Provides a dashboard for server, terminal, logs, email, and integrations
- Keeps Hermes Agent online without users managing infra
- Packages the open-source agent experience into a simple managed product

## Public Repo

[hermes-box](https://github.com/trywraithprotocol/hermes-box) contains the public site and product dashboard code for Hermes Box.

## Stack

Next.js, React, TypeScript, Tailwind CSS, Supabase, Firebase, Solana wallet tooling, and Vercel.
